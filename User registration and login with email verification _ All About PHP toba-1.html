User registration and login with email verification

January 26, 2016December 24, 2016 Kamal MY SQL, PHP

In this tutorial, we will look into Login Registration with Email Verification, Forgot Password Using PHP.
When running this code in local machine (localhost) email sending function won’t work also email verification needs a domain and a server to capture the call back from the user when they click on the link. Simply host this code in a real server in order to try out the full functionality.

Overview

For this project, we need 7 files in order to complete the task. So, let’s create the files. You need to keep all the files in same folders.

  • Configdb.php
  • Confirm.php
  • Index.php
  • Login.php
  • Logout.php
  • Home.php
  • Register.php

Database Schema

Let us start by creating a database needed to store user credentials. I will be naming my database “userreg”. You can name your’s anything you like. But something relevent to the project would be good.
Then add the necessary tables to the database. You can use the code snippet below to add tables to the database.

CREATE TABLE `user` (
`id` INT( 50 ) NOT NULL AUTO_INCREMENT ,
`username` VARCHAR( 50 ) NOT NULL ,
`email` VARCHAR( 100 ) NOT NULL ,
`password` VARCHAR( 20 ) NOT NULL ,
`com_code` VARCHAR( 255 ) default NULL,
PRIMARY KEY ( `id` )
) ENGINE = InnoDB

Configdb.php

This file contains all the configuration that is needed in order to establish a successful connection with the database.
If the user intendeds to change the SQL properties he/she can change the following attributes in order to make the application work.

<?php
$mysqli=mysqli_connect('localhost','root','','userreg') or die("Database Error");
?>

Confirm.php

This file contains the code that deletes the key relevant to the user in the database when a user clicks on the confirmation link sent via mail.
When the user redirects to this page it extracts the key from a user and issues a query which will search the key and delete it from the database. So the user gets confirmed.
In other terms, it’s a page that confirms the user registration.

<?php
include('configdb.php');
$passkey = $_GET['passkey'];
$sql = "UPDATE user SET com_code=NULL WHERE com_code='$passkey'";
$result = mysqli_query($mysqli,$sql) or die(mysqli_error());
if($result)
{
echo '<div>Your account is now active. You may now <a href="login.php">Log in</a></div>';
}
else
{
echo "Some error occur.";
}
?>

Register.php

This file contains the code which is used for user registration also the email sending code is also available in the following code segment. In order to test users will need to edit this file replacing their domains instead of http://www.exampleDomain.com/confirm.php?passkey=$com_code”

<?php
session_start();
include('configdb.php');
if(isset($_POST['submit']))
{
//whether the username is blank
if($_POST['username'] == '')
{
$_SESSION['error']['username'] = "User Name is required.";
}
//whether the email is blank
if($_POST['email'] == '')
{
$_SESSION['error']['email'] = "E-mail is required.";
}
else
{
//whether the email format is correct
if(preg_match("/^([a-zA-Z0-9])+([a-zA-Z0-9._-])*@([a-zA-Z0-9_-])+([a-zA-Z0-9._-]+)+$/", $_POST['email']))
{
//if it has the correct format whether the email has already exist
$email= $_POST['email'];
$sql1 = "SELECT * FROM user WHERE email = '$email'";
$result1 = mysqli_query($mysqli,$sql1) or die(mysqli_error());
if (mysqli_num_rows($result1) > 0)
{
$_SESSION['error']['email'] = "This Email is already used.";
}
}
else
{
//this error will set if the email format is not correct
$_SESSION['error']['email'] = "Your email is not valid.";
}
}
//whether the password is blank
if($_POST['password'] == '')
{
$_SESSION['error']['password'] = "Password is required.";
}

//if the error exist, we will go to registration form
if(isset($_SESSION['error']))
{
header("Location: index.php");
exit;
}
else
{
$username = $_POST['username'];
$email = $_POST['email'];
$password = $_POST['password'];
$com_code = md5(uniqid(rand()));

$sql2 = "INSERT INTO user (username, email, password, com_code) VALUES ('$username', '$email', '$password', '$com_code')";
$result2 = mysqli_query($mysqli,$sql2) or die(mysqli_error());

if($result2)
{
$to = $email;
$subject = "Confirmation from TutsforWeb to $username";
$header = "TutsforWeb: Confirmation from TutsforWeb";
$message = "Please click the link below to verify and activate your account. rn";
$message .= "http://www.exampleDomain.com/confirm.php?passkey=$com_code";

$sentmail = mail($to,$subject,$message,$header);

if($sentmail)
{
echo "Your Confirmation link Has Been Sent To Your Email Address.";
}
else
{
echo "Cannot send Confirmation link to your e-mail address";
}
}
}
}
?>
